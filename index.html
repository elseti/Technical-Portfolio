Step Step Revolution

A memory-based arcade game created with FPGA.

Step Step Revolution

Project:  Computation Structures

Languages / Frameworks:  Lucid, FPGA, Alchitry

Preview :  SUTD Page

Team Members:  Lee Jing, Joel Ng, Tan Lay Lin, Suthar Prachi Jayesh, Eliana Setiabudi, Devanshi Joshi, Ankita Parashar, Andrean Priadi

Repository :  Github

- Led 7 team members in creating a physical arcade game, using FPGA and Lucid to develop architecture and circuitry.
- Formulated the data path, coded game functionalities and ALU, integrated hardware, and debugged hardware faults.

Step Step Revolution intends to test its players on both their memory and coordination skills with the goal being to clear as many rounds as possible. To do so, a sequence of LED lights will light up and the player must memorise them and re-enter them into the machine within 30 seconds to advance to the next round. There are a total of 10 such rounds. The project was made with 5 LED arcade buttons, a single plastic button, 2 4 digit 7-segment displays and an Alchitry Au FPGA. The rest of the materials excluding those used for wired connection were repurposed from other scrap material.

Totallia: Ghidra Plugin

A working Ghidra plugin for easier software reverse engineering.

Totallia: Ghidra Plugin

Project :  Ghidra Plugin

Languages:  Python/Jython, angr, Assembly

Link :  Github

Team Members:  Caleb Ci, Daniel Chamorro Quillupangui, Jarrett Kinsey, Michael Pham, Alfonso Gonzalez, Eliana Setiabudi, Chun Hui Wee

Totallia is a Ghidra plugin which focuses on improving user experience in code reversal. It employs a VirusTotal script which uses an external API to detect existing malware present in the binary file. We improved the existing scripts to create a tool that improves user experience, includes useful functionalities, and protects the user by adding an extra layer of security.

Furthermore, by displaying the inputs angr is checking for, we introduced a feature to observe how it was operating. We also added a feature to export the settings so users can update the angr script outside of Ghidra. By doing this, users can create more advanced scripts, which are occasionally required to reverse particular binaries.

BrainHack comprised a series of exciting activities, and one of them was the Cyber Defenders Discovery Camp (CDDC). At the camp, participants were able to delve into topics within the exciting field of cybersecurity, including cyber forensics and cryptography.
